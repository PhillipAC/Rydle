Rules:

Rydle is a game where you try to guess what year an event took place.
In normal play the event will take place some year on this day in history.
In random play the event will be any event in the past

For each guess each digit of the year will get one of three results.

Green: The digit is correct
Yellow: The digit is in the incorrect place
Black: The digit is not in the answer

Once you are down to one unknown digit an additional hint will be given.

↑: The event happened after the guess
↓: The event happened before the guess

Try to find the year the event took place as quick as you can!

All events are pulled from the Wikimedia API and are not picked by the creators of Rydle.
